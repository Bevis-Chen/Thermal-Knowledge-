所謂壓力差，簡單的公式解釋 ∆P = P1 - P2，
那什麼是壓力? 
基本解釋為: 每個單位面積所承受的力. 也就是 P = F/A
每個單位面積的對象，就是每個受力的物體. 會是大大小小的散熱器、也會是粒子分子的個體. 
# 其實要討論的是壓力是針對誰> 博努利定理
## 流體靜壓 + 流體動壓 + 流體位能  = 總壓
當流體受到 
### 做功 (功 = 作用力 * 作用距離)，
內部原本總壓為常數(也就是平衡，每個流體分子互相亂撞但是能量平衡.
假設有一個氣球充飽氣體且沒有破洞. 此時對於內部氣體來說沒有位能也沒有總動能，所以總壓只剩下靜壓.
也就是空氣內部分子互相撞擊且維持宏觀平衡. 總壓不為零但是大於1大氣壓. 如果不平橫氣球就爆了.
被針刺破的一瞬間: 氣球內部平衡的空氣被打破平衡，因為氣球外是大氣壓力。
所以氣球內的高壓與氣球外相對於氣球內低壓的空氣連通. 
最後空氣由高壓流向低壓: 氣球被刺破後便萎縮了.)

應用到風扇，風扇在一大氣壓下開放空間轉動. 
  - 此時風扇的葉片推動空氣> 出風口空氣首先被推出> 
  - 此刻風扇腔內(含葉片)的空氣所承受壓力變小(也沒有空氣了)>
  - 入風口空氣承受壓力相對於風扇內的空氣所承受壓力較大>
  - 入風口空氣被扇葉捲入> 
  - 空氣從入風口進入>
  - 進入風扇腔內進入經過扇葉> 
  - 通過出風口.
